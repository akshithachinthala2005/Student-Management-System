\# Student Management System (MySQL)



\## Project Overview

This project is a Student Management System developed using MySQL. It manages student information, courses, faculty, enrollments, and marks. The project demonstrates database design, SQL queries, joins, and data analysis.



\## Features

\- Create a Student Management database

\- Create relational tables

\- Insert sample data

\- Perform basic SQL queries

\- Perform analysis queries using JOIN, GROUP BY, and aggregate functions



\## Database Tables

\- Students

\- Courses

\- Faculty

\- Enrollments

\- Marks



\## Files

\- 01\_create\_database.sql

\- 02\_create\_tables.sql

\- 03\_insert\_data.sql

\- 04\_basic\_queries.sql

\- 05\_analysis\_queries.sql



\## Basic Queries

\- View all students

\- View all courses

\- Display faculty details

\- Students with marks greater than 80

\- Students born after a specific date



\## Analysis Queries

\- Total students enrolled in each course

\- Average marks of students

\- Top scoring students

\- Faculty handling maximum students

\- Students scoring above average



\## Technologies Used

\- MySQL

\- MySQL Workbench
